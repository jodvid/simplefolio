Smart Brain Project

I built a Facial Recognition app using React framework and Node.js at the back end with PostgresDB as its database. The most challenging part as new developer when connecting the frontend and the server and will handle any requests from the app, including user registration, user data change or image processing. To detect faces in the images, I used the Face Detection Model provided by the Clarifai API. This service uses artificial intelligence to recognize the visual content. The user submits an image, and the coordinate locations of where the faces might appear are returned.
